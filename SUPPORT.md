# Support

If you encounter issues with this sample:

1. Check the [Known Issues](docs/KNOWN_ISSUES.md) for documented problems and solutions.
2. Search existing [issues](../../issues) to see if the problem has been reported.
3. If not, open a new issue with:
   - A clear description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment (OS, Python version, Node version, AWS region)

This is a sample project and not an officially supported AWS service. Response times may vary.
